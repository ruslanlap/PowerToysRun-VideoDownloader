### üöÄ Quick Start
1. Download the ZIP file for your platform (x64 or ARM64)
2. Extract to `%LOCALAPPDATA%\Microsoft\PowerToys\PowerToys Run\Plugins\`
3. Restart PowerToys
4. Start using with `Alt+Space` then type `dl <URL>`
